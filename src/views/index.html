Getting Started

Installation

To get started with the Weather API, clone this repository and install the required dependencies:


    git clone https://github.com/kathulhur/panahon-api.git
    cd panahon-api
    npm install

Usage

  1. Rename the .env.example file to .env and provide the necessary environment variables.
  2. Run the following command to start the API server:
npm start

By default, the API will be available at http://localhost:3000.

Endpoints

Register

Register a new user.

  • URL: /api/auth/register
  • Method: POST
  • Request Body:
{
  "name": "exampleuser",
  "email": "exampleuser@mail.com",
  "password": "password123"
}

Success Response:

  • Status Code: 201 (Created)
  • Response Body:

  {
      "message": "Registration successful",
      "status": 200,
      "data": {
          "name": "exampleuser",
          "email": "exampleuser@mail.com"
      }
   }

Login

Get Weather

Retrieve weather information for a specific city.

  • URL: /api/weather/:city
  • Method: GET
  • Success Response:

    {
        "data": {
            "location": {
                "name": "Manila",
                "region": "Manila",
                "country": "Philippines",
                "localtime": "2023-05-29 16:19"
            },
            "current": {
                "temp_c": 32,
                "condition": {
                    "text": "Partly cloudy",
                    "icon": "//cdn.weatherapi.com/weather/64x64/day/116.png"
                }
            }
        },
        "message": "OK",
        "status": 200
     }

User Management [Protected Routes]

  • GET /api/users: Retrieves all users (only accessible to admin users).
  • POST /api/users: Creates a new user (only accessible to admin users).
  • PUT /api/users/:id: Updates user information (only accessible to admin users).
  • DELETE /api/users/:id: Deletes a user (only accessible to admin users).
